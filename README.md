# Netflix Data Analysis Using Python and Pandas

## Project Overview

This project performs Exploratory Data Analysis on a Netflix titles dataset containing movies and TV shows.  
The goal is to analyze content distribution, genre trends, country-wise contribution, yearly growth, and release patterns using Python, Pandas, Matplotlib, and Seaborn.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

The dataset contains information about Netflix titles including:

- Title
- Type
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre

## Key Metrics

- Analyzed 7K+ Netflix titles
- Explored 20+ genres
- Analyzed content across 50+ countries
- Calculated year-over-year content growth
- Created 10+ visualizations for better interpretation

## Analysis Performed

1. Data loading and inspection
2. Missing value handling
3. Date column conversion
4. Feature engineering
5. Movie vs TV Show comparison
6. Genre analysis
7. Country-wise analysis
8. Rating distribution
9. Year-wise content growth
10. Release year trend analysis

## Key Insights

- Netflix has more movies than TV shows in the dataset.
- International Movies, Dramas, and Comedies are among the most common genres.
- The United States and India are among the top content-producing countries.
- Netflix content increased significantly in recent years.
- Mature audience ratings appear frequently in the catalog.
